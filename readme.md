To run this code, you should run these two commands in the terminal 

make
make run 

the first command ( make ) will compile the code according to the Makefile we made  and make all the executable files to run afterwards 
the 2nd commande ( make run ) will run the files according to the Makefile and start the code 

ps  : after running the code you can run a third command 
make clean 

and this oommand (make clean) will remove all the executable files it made
